# Filament Manager (Python)

A simple command-line application to manage 3D printing filament.

## 📌 Features

- Add new filament (name, material, weight, price per kg)
- View all stored filaments
- Track filament usage
- Calculate material cost based on usage
- Data is saved in a JSON file for persistence

## 🛠️ Technologies

- Python 3
- JSON for data storage

## 🚀 How to run

1. Clone or download this repository
2. Open the project folder in VS Code or terminal
3. Run the program:

```bash
python3 main.py
